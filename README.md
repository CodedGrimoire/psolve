# psolve

A small TypeScript practice file with 10 classic problem-solving functions.

## File

- `solution.ts`: contains standalone utility/problem functions (string, arrays, math, and sequence tasks).

## Included Functions

- `reverseString(s: string): string`
- `fizzBuzz(n: number): string[]`
- `findLargest(nums: number[]): number`
- `isPalindrome(s: string): boolean`
- `sumArray(nums: number[]): number`
- `countVowels(s: string): number`
- `factorial(n: number): number`
- `getEvenNumbers(nums: number[]): number[]`
- `fibonacci(n: number): number[]`
- `findMinimum(nums: number[]): number`

## Run

If you have `ts-node` installed:

```bash
ts-node solution.ts
```

Or compile and run with TypeScript:

```bash
tsc solution.ts
node solution.js
```

Note: the file currently defines functions only (no `console.log` test calls).
